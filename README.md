naskpass
========

Ncurses based replacement for askpass (related to cryptsetup).

dependencies
========
post-build: libncurses5, libtinfo5
pre-build: libncurses5-dev, libtinfo-dev

note
========
Conflicts: plymouth
Plymouth may not like naskpass (never verified).
